Topic 3 · Sections 3.1 – 3.4

3.1 Backing Up Data

Backing Up — copying files or data to a different medium in case of damage or loss of the original.

Why Back Up Data?

  • To safeguard against loss of data
  • To safeguard against damage caused by hackers
  • In case files need to be sent elsewhere, to protect the original file format
  • Backups can be stored online for easy access from anywhere

Types of Backing Storage

Magnetic

Data held magnetically on a magnetised surface. Includes tapes and hard disk drives.

Optical

Data burned as 'dots' onto surface with a laser. Includes CDs, DVDs, Blu-ray.

Solid-State

No moving parts. Includes SSDs, pen drives, flash memory cards.

3.2 Magnetic Storage — Tape Drives

Surface area of media is magnetised to hold bits of data. Read/write device reads bits on the surface.

Magnetic Tape Drives

Uses: batch processing applications; backup of vast amounts of data; long-term archiving (very stable, huge capacity)

Advantages:

  • Generally less expensive than hard disk drives
  • Very robust technology
  • Huge data storage capacity
  • Fast data transfer rate

Disadvantages:

  • Prolonged data access times — uses serial access
  • Updating requires a new tape to store the final version
  • Affected by magnetic fields — strong fields can corrupt data

Access Methods

Serial Access (Tape):

  • Must read through all data before reaching required data
  • Slow for random access
  • Good for sequential batch processing

Direct/Random Access (Disk):

  • Can go directly to any data location
  • Much faster for retrieving specific files

Magnetic Hard Disk Drives

Fixed Hard Disk Drive

Uses: store OS, system software and working data; application files; real-time and online systems; file servers for networks

Advantages:

  • Very fast data transfer rate and access times
  • Very large memory capacity

Disadvantages:

  • Can be fairly easily damaged
  • Many moving parts reduce reliability
  • Read/write operation can be noisy vs SSD

Portable Hard Disk Drive

Uses: backup to prevent data loss; transfer data/files/software between computers

Advantages:

  • Fast data access time and transfer rate
  • Very large memory capacity
  • Portable method of transferring information between computers

Disadvantages:

  • Easily damaged if dropped or incorrectly shut down
  • Data transfer rate not as fast as fixed HDD

3.3 Optical Storage Media

Data burned as 'dots' onto surface using a high-powered laser. Lower-powered laser reads the data back.

Blu-ray vs DVD

Three key differences between Blu-ray and DVD:

Blue Laser

Blu-ray uses a blue laser (405 nm) vs red laser (650 nm) on DVD. Smaller wavelength = smaller pits and lands = 5× more storage than DVD

Encryption

Blu-ray discs automatically include a secure encryption system, helping prevent piracy and copyright infringement

Data Transfer Rate

DVD: 10 Mbps
Blu-ray: 36 Mbps — much faster data transfer and access speed

Pen Drives & Flash Memory Cards

Pen Drives (Memory Sticks)

Uses: transporting files between computers; backing store; security device (dongle) to prevent software piracy

Advantages:

  • Very compact and portable
  • Very robust
  • No additional software needed on most computers
  • Not affected by magnetic fields

Disadvantages:

  • Cannot make data 'read-only' to protect it
  • Easy to lose due to small size
  • Incorrect removal corrupts data and makes it useless

Flash Memory Cards

Types: SD (secure digital — widely used) · xD (extreme digital — older cameras) · CFast (compact fast — professional cameras)

Uses: digital camera photos; mobile phone memory; MP3 players; hand-held devices

Advantages:

  • Very compact; durable (no moving parts)
  • Can hold large amounts of data
  • Cameras and smartphones can read/write them

Disadvantages:

  • Expensive per GB vs HDDs
  • Lower capacity than hard disks
  • Finite read/write lifespan
  • Very small — easily lost/stolen/damaged
  • Often requires card reader/USB converter

Topic 3 — Key Points

Magnetic

  • Tapes: serial access, archiving, batch
  • Fixed HDD: fast, large, OS storage
  • Portable HDD: backup, file transfer

Optical

  • CD/DVD-R: write once, cheapest
  • CD/DVD-RW: rewritable, backup
  • Blu-ray: largest, fastest, encrypted

Solid-State

  • SSD: fastest, reliable, no moving parts
  • Pen Drives: portable, compact transport
  • Flash Cards: cameras, phones, media
Always choose the storage type based on the need: speed (SSD), capacity (HDD), portability (pen drive), archiving (tape), or low cost (optical).
