Types & Components of
Computer Systems

1.1 Hardware & Software

Hardware — the electronic, physical components of a computer system

Internal Hardware

  • Central Processing Unit CPU
  • Motherboard
  • Internal memory — RAM & ROM
  • Graphics & Sound Cards
  • Storage: HDD / SSD
  • Network Interface Card NIC

External Hardware

  • Monitor
  • Keyboard
  • Mouse
  • Printer
  • External Storage Devices

Software

Software — the techniques, methods, commands and instructions that tell a device what to do and how to do it; the programs which control the computer and process data.

Application Software

  • Provides services for specific user tasks
  • General-purpose or custom-made (specialist)
  • E.g. spreadsheets, databases, word processors

System Software

  • Compilers — translate high-level language to machine code
  • Linkers — create links between language processors
  • Device Drivers — enable external devices to work
  • Utilities — antivirus, scan disk, defragmentation
  • Operating Systems — platform for all other software

Analogue & Digital Data

Computers only understand digital data — binary digits (0 and 1). Real-world data is analogue — not limited to two values.

  • Analogue data forms smooth curves
  • Digital data changes in discrete steps
  • ADC — Analogue-to-Digital Converter: converts real-world signals for the computer
  • DAC — Digital-to-Analogue Converter: converts output back to analogue form
Analogue (smooth)
Digital (steps)

Real-world data → ADC → Computer processes → DAC → Output

1.2 Central Processing Unit (CPU)

The component that understands and executes hardware and software operations — also called a microprocessor.

Three Key Parts

  • Control Unit (CU) — manages input & output devices; controls flow of data
  • ALU — Arithmetic & Logic Unit; performs calculations and logical decisions
  • Registers — tiny, ultra-fast memory regions inside the CPU

CPU Role

Receives input → processes instructions → produces output

Present in most PCs and also in household gadgets where control or monitoring is required.

1.3 Operating Systems

Special software that enables computer systems to function and allows users to communicate with them.

OS Functions

  • Controls operation of input, output & backing storage
  • Supervises loading, running & storage of programs
  • Deals with errors in application programs
  • Maintains security of the whole system
  • Maintains a computer log

User Interface Types

  • CLI Command Line Interface
  • GUI Graphical User Interface
  • Dialogue-based Voice commands
  • Gesture-based Movement input

Command Line Interface CLI

User types commands directly to communicate with the computer. Mainly used by programmers who need direct communication.

Advantages

  • User in direct communication with computer
  • Not restricted to pre-defined options
  • Can alter computer configuration settings

Disadvantages

  • Must learn many commands for basic operations
  • Time-consuming — all commands typed manually
  • Very error-prone — commands need correct format, spelling

Graphical User Interface GUI

Uses WIMP — Windows, Icons, Menus, Pointing device. Mainly used by end-users.

Advantages

  • No need to learn commands
  • More user-friendly — icons represent applications
  • Click on icon to launch — simpler than typing

Disadvantages

  • Uses considerably more computer memory than CLI
  • User limited to icons provided on screen
  • Needs a more complex operating system

Dialogue & Gesture-Based Interfaces

Dialogue-Based — Voice Commands

  • No need to take hands off steering wheel
  • Useful for people with disabilities
  • Can be used as a security/voice recognition feature
  • Disadvantages: still unreliable; background noise causes errors; complex to setup; limited commands

Gesture-Based — Movement Input

  • Replaces mechanical input devices
  • No physical contact required — very natural interface
  • No training needed
  • Disadvantages: unintentional movement picked up; only works within ~1.5 m of sensor; limited gestures recognised

1.4 Desktop Computers

General-purpose computer made of separate monitor, keyboard, mouse and processor unit — not very portable.

Uses: office & business, education, entertainment, gaming

Advantages

  • Easier and less costly to upgrade/expand
  • Standardised spare parts — cheaper
  • Better specifications for a given price
  • No critical power consumption — plugs into wall
  • Fixed position — less likely to be stolen/damaged
  • Wired connections — more stable internet

Disadvantages

  • Not portable — made of separate components
  • Wiring clutters desk space
  • Files must be copied to portable media to work elsewhere

Laptop Computers

All components (monitor, keyboard, pointer, processor) in one unit — extremely portable. Lightweight, low power, low heat output.

Uses: office & business, education, gaming, entertainment

Advantages

  • Portable — all components in one unit
  • No trailing wires or desk clutter
  • Takes up less desk space; usable in public
  • Takes advantage of Wi-Fi features

Disadvantages

  • Portability makes it easier to be stolen
  • Limited battery life — must carry charger
  • Keyboard/pointing devices can be awkward
  • Not easy to upgrade (e.g. adding more RAM)

Smartphones

Phones with an OS (iOS, Android) allowing computer applications (apps). Connect via Wi-Fi or 3G/4G/5G.

Uses: emails, apps, camera/video, music/video player, GPS, banking, VoIP, social media

Advantages

  • Very small & lightweight — portable
  • Internet on the move
  • Usable almost anywhere (Wi-Fi + mobile networks)
  • GPS/location apps not available on desktop/laptop
  • Reasonable battery life

Disadvantages

  • Small screen & keyboard — difficult to read/type
  • Browsing & camera drain battery quickly
  • Limited memory vs laptops/desktops
  • Not all websites compatible with smartphone OS
  • Easier to lose or be stolen
  • Mobile network can be slower than Wi-Fi

Tablets & Phablets

Tablets

Similar to smartphones but larger. Uses: emails, apps, camera, media player.

Advantages vs Laptops:

  • Very fast to switch on — no OS loading delay
  • Fully portable and lightweight
  • Touch screen — simple, no extra input devices
  • Low heat output — solid-state technology
  • Longer battery life; stays connected in standby

Disadvantages vs Laptops: more expensive, limited storage, slow touch-screen typing, fewer file formats supported

Phablets

Hybrid between a tablet and smartphone — larger screen than a phone but smaller than a tablet.

Additional Uses:

  • Entertainment — streaming music, video, TV
  • Gaming including group games
  • Camera & video
  • Internet, social networks, QR codes
  • GPS navigation
  • Telephone banking
  • VoIP & video calling
  • Instant messaging & social contact
  • Remote device control

1.5 Emerging Technologies — Artificial Intelligence

AI — a machine or application that carries out tasks requiring some degree of intelligence, such as:

AI Applications

  • Use of language & face recognition
  • Operating machinery — cars, aeroplanes, trains
  • Weather forecasting from data analysis
  • Driverless (autonomous) vehicles
  • Robotics for amputees & disability assistance
  • Bomb disposal & dangerous tasks

Negative Impacts of AI

  • Unemployment — robots do tedious human tasks with greater efficiency
  • Dependency on technology — could cause future problems
  • De-skilling — tasks once considered talents are now routine machine processes

Extended Reality XR

Combined real and virtual environments. Two main types:

Augmented Reality AR

Merger of virtual and physical world — lays digital objects onto real ones. Example: Pokémon GO!

Impacts:

  • Safety & rescue operation training
  • Shopping/retail — better product preview
  • Healthcare — better understanding of patient's body

Virtual Reality VR

Creates an entirely new virtual world — no link to actual surroundings. Requires VR headset, joystick & headphones.

Impacts:

  • Military applications & training
  • Education — viewing ancient buildings
  • Healthcare & surgery simulation
  • Media — interactive special effects
  • Scientific visualisation — molecular structures/cells
