Input & Output Devices

Topic 2 · Sections 2.1 – 2.3

Topic 2 Overview

Input Devices

Any hardware that allows a user to enter data or instructions into a computer.

  • Can send data — cannot receive data
  • Necessary for receiving commands & data to process
  • Under the user's control or direct data entry

Output Devices

Any hardware that presents processed data to the user in a human-readable format.

  • Can receive data — cannot send data
  • Displays or produces the result of processing
  • E.g. words on a printer, graphics on a monitor

Keyboard & Numeric Keypad

Keyboard

Uses: data input into applications; typing commands

Advantages:

  • Fast text entry; well-known method
  • Easy verification — compare source with typed data

Disadvantages:

  • Difficult if user has limited arm/wrist use
  • Slow compared to direct data entry
  • Large device — takes up desk space

Numeric Keypad

Uses: ATMs (PIN entry), POS terminals (manual barcode entry), chip & PIN devices, numeric spreadsheet data

Advantages:

  • Faster than keyboards for numeric data
  • Small & portable

Disadvantages:

  • Small keys can make input difficult
  • Number order can be unintuitive

Pointing Devices

Mouse

Uses: opening/closing software; file management; image editing; pointer control

Adv:

  • Faster than keyboard for options
  • Quick navigation
  • Less desk area than keyboard

Dis:

  • Difficult for restricted hand movement
  • Clogs with dirt; needs flat surface

Touchpad

Uses: same as mouse — integrated into laptops

Adv:

  • No separate mouse needed — aids portability
  • Usable without flat surfaces

Dis:

  • Difficult with limited hand movement
  • Harder pointer control than mouse
  • Drag & drop more complicated

Tracker Ball

Uses: alternative for RSI sufferers; industrial control rooms; luxury cars

Adv:

  • Less fine control needed; easier with wrist problems
  • More accurate; more robust; less desk space

Dis:

  • Not supplied as standard — more costly
  • Users may need training

Remote Control, Joystick & Driving Wheel

Remote Control

Uses: TVs, Blu-ray players, Hi-Fi systems; industrial process control

Adv:

  • Operable from a distance
  • Control from safe distance in hazardous environments

Dis:

  • Difficult with limited hand movement
  • Physical obstacles block signal

Joystick

Uses: video games; simulators mimicking actual controls

Adv:

  • Easier than keyboards for screen navigation
  • More realistic control in some applications

Dis:

  • On-screen pointer more difficult to control

Driving Wheel

Uses: car racing games; vehicle simulators

Adv:

  • More realistic than mouse or joystick
  • Easier steering control

Dis:

  • Expensive compared to mouse/joystick
  • Steering over-sensitive; feedback non-existent in basic models

Touch Screen, Scanner & Digital Camera

Touch Screen (Input)

Uses: ATMs, POS terminals (restaurants), interactive whiteboards, CBT testing

Adv:

  • Faster than keyboard/mouse for options
  • Very easy; no training needed
  • Can expand display size

Dis:

  • Limited input options
  • Frequent use → RSI, arm muscle strain
  • Screen gets dirty

Scanner

Uses: converting documents/photos; recovering damaged docs; barcode scanning at POS

Adv:

  • Images stored for editing
  • With OCR: faster & more accurate than typing
  • Can recover damaged documents

Dis:

  • Quality limited by scanner resolution
  • Fairly slow, especially in colour mode

Digital Camera

Uses: photography; car reversing aid; dentistry; virtual reality tours

Adv:

  • Better quality than traditional cameras
  • Fast upload to computer; no film needed
  • Memory cards store thousands of photos; delete unwanted

Dis:

  • Requires computer literacy
  • Images need compressing to save memory

Microphone, Sensors & Light Pen

Microphone

Uses: voice recognition; intruder alarm sensor; audio conferencing; VoIP

Adv:

  • Faster to speak than type
  • Real-time sound manipulation
  • Improves security via authentication

Dis:

  • Sound files use large amounts of memory
  • More error-prone than typing

Sensors

Types & Uses: Temperature (washing machines, ovens) · Pressure (alarms) · Light (street lighting) · Sound/acoustic (intruder alarms) · Humidity (glasshouses) · pH (chemical processes)

Adv:

  • More accurate than humans
  • Continuous monitoring; instant response
  • No human intervention needed

Dis:

  • Faulty sensors give spurious results
  • Most are analogue → require ADC

Light Pen

Uses: selecting objects on CRT screens; drawing on screens

Adv:

  • Greater accuracy than touch screens
  • Small & portable
  • Easy to use

Dis:

  • Lagging when drawing
  • Only works with CRT monitors
  • Outdated technology

Magnetic Stripe vs Chip & PIN vs Contactless

Magnetic Stripe

Uses: Credit/debit cards at ATMs; security entry to buildings and hotels

Advantages:

  • Faster than keying data manually
  • Error-free — no human intervention
  • Secure — data not human-readable
  • Robust — no moving parts

Disadvantages:

  • Data lost if card is damaged
  • Doesn't work at a distance

Chip & PIN

Uses: Card payments at shops, restaurants, ATMs

Advantages:

  • More secure than contactless — PIN is second security layer
  • More robust than magnetic stripe

Disadvantages:

  • PIN must be entered carefully — risk of shoulder surfing

Contactless Card

Uses: Quick payments without entering PIN

Advantages:

  • No incorrect PIN worry
  • Eliminates shoulder surfing
  • Faster transaction; encryption with unique transaction code

Disadvantages:

  • Spending limit applies
  • More expensive to implement
  • Thief can intercept transaction using a suitable reader

RFID, OMR, OCR, Barcode & QR

RFID Reader

Uses: livestock tracking; retail (instead of barcodes); admission passes; library books

Adv: No line-of-sight needed; very reliable; fast; bidirectional; bulk detection possible

Dis: Tag collision; easy to jam/hack radio waves; more expensive than barcode

OMR

Uses: MCQ papers; voting papers; survey forms

Adv: Very fast; accurate; no human intervention; more accurate than OCR

Dis: Complex form design; problems require manual checking

OCR

Uses: passports; identity cards; converting hard copies; ANPR; digitising archives

Adv: Much faster than manual typing; no human intervention

Dis: Difficulty reading different handwriting; still not very accurate

Barcode & QR Scanners

Barcode Uses: supermarket stock control; libraries; electrical equipment checks

Barcode Adv: Faster than manual; tried and trusted; automatic stock control

Barcode Dis: Expensive; barcodes can be swapped; easily damaged

QR Uses: advertising; Wi-Fi auth; restaurant menus

QR Adv: Holds more data; fewer errors; readable by smartphones; can be encrypted

QR Dis: Multiple formats exist; can transmit malicious data

2.3 Output Devices — Monitors

CRT Monitors (Cathode Ray Tube)

Uses: complex diagrams in CAD; used with light pens

Advantages:

  • Visible at wider range of angles than LCD
  • Compatible with light pens

Disadvantages:

  • Very heavy and bulky
  • Run very hot — fire risk
  • Use much more power than LCD
  • Flickering causes headaches
  • Image burn-in is common

LCD Screens (Liquid Crystal Display)

Uses: main output device for modern computers; mobile phones; laptops; video games

Advantages:

  • Very efficient — low power consumption
  • Lightweight; available in various sizes
  • No flickering; very sharp resolution

Disadvantages:

  • Colour and contrast inconsistent at different angles
  • Motion blur is common
  • Lower contrast than CRT

Touch Screen (Output) & Multimedia Projector

Touch Screen (as Output)

Uses: ATMs; POS terminals; information kiosks

Advantages:

  • No additional input devices needed
  • Very interactive — swiping and gestures
  • High-tech feel to devices

Disadvantages:

  • Gets dirty with frequent use
  • Frequent use → RSI and arm muscle strain
  • Not very accurate or fast for large data input/output

Multimedia Projector

Uses: training presentations; advertising; home cinema

Advantages:

  • Many people can watch without crowding around a screen
  • Avoids need for several networked computers

Disadvantages:

  • Images can be fuzzy
  • Expensive to purchase
  • Setup can be complicated

Speaker & Actuator

Speaker

Uses: all phones and most computers; assisting visually impaired; playing downloaded sound files

Advantages:

  • Amplified sounds can be very loud — heard by a large crowd
  • Simple technology
  • Especially useful for blind people — reading e-books

Disadvantages:

  • Output can disturb others in the same area (e.g. office)
  • High sound quality only from expensive speakers
  • Take up desk space

Actuator

Uses: turning on motors, buzzers and switches; allowing a computer to control physical devices

Advantages:

  • Allows remote operation of many devices
  • Relatively inexpensive

Disadvantages:

  • Additional device in the system — another potential point of failure
  • Analogue devices — signals need converting using a DAC

Topic 2 — Key Points

Input Devices

  • Keyboard — general data/text entry
  • Mouse/Touchpad/Trackerball — pointer control
  • Touch Screen — fast option selection
  • Sensors — continuous automated monitoring
  • Microphone — voice input & authentication

Direct Data Entry

  • Magnetic Stripe / Chip & PIN / Contactless — card payments
  • RFID — no line-of-sight tracking
  • OMR / OCR / Barcode / QR — automated recognition

Output Devices

  • CRT — older; wide viewing angle; bulky
  • LCD — modern; energy-efficient; sharp
  • Laser Printer — fast; high-volume; quality
  • Inkjet — quality; affordable; low volume
  • Dot Matrix — carbon copies; noisy
  • 3D Printer — custom physical objects
  • Actuator — computer controls physical devices
